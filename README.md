# やつかほバス時刻表

## 目的

金沢工業大学とやつかほキャンパスを繋ぐ、バスの時刻表が(めちゃくちゃ)不便なので Web ページを作りました。

やつかほリサーチキャンパスの[公式ページ](https://www.kanazawa-it.ac.jp/about_kit/yatsukaho.html)

## 注意事項

Web ページは[こちら](https://101ta28-yatsukaho-bus.netlify.app)
